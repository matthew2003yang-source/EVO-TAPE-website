### How Claude will use them
- `sixpack-box.jpg` → **EVO TAPE Original 6 Pack** product images (card, PDP main + thumbs)
- `original-set.jpg` → **EVO TAPE Original** product images (cropped: front box, angle, roll, back)
- `james-model.jpg` → hero / about / applications-hero lifestyle model shots (cropped per pose)
- `applications-grid.jpg` → the 8 application cards (shoulder, wrist, back, knee, calf, thigh, ankle, neck), cropped from the grid

> The application cards need body-part-with-tape photos. The only ones available are inside
> the Applications screenshot, so Claude will crop those. If you have the original
> high-res application photos, drop them in here too (named `app-shoulder.jpg`,
> `app-wrist.jpg`, etc.) and Claude will use those instead for much sharper results.

Tip: in the Claude desktop app you can usually drag an image out of the chat onto your
desktop, then move it into this folder — or right-click → Save Image As.
